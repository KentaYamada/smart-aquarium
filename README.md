# Smart aquarium
水槽の水温、水質を管理するアプリケーション

## Features
- Backend
    - Python3.x
    - Flask
- Frontend
    - node (v12.13.1)
    - npm (v6.12.1)
    - TypeScript(v3.5.3)
    - SCSS(v4.12.0)
    - Vuejs (cli v4.0.5)
    - Buefy
    - Bulma

## Development
### Start up local server
### Frontend
``` sh
cd /path/to/flask-vuejs-startup-kit/frontend
npm run serve
```

### Backend
``` sh
cd /path/to/flask-vuejs-startup-kit
python3 -B dev.py
```
